# Pizza Cart 

Интерактивное приложение для заказа пицц с корзиной, сортировкой и сохранением данных.



https://pizza-nine-black.vercel.app

##  Описание

 Приложение для выбора пицц. Можно добавлять пиццы в корзину, менять количество, сортировать позиции по цене и количеству. 

##  Функционал

- ✅ Добавление пицц в корзину
- ✅ Увеличение/уменьшение количества
- ✅ Автоматическое удаление при количестве 0
- ✅ Сортировка по умолчанию / по цене / по количеству
- ✅ Подсчёт общей суммы
- ✅ Сохранение корзины в localStorage
- ✅ Чистый и отзывчивый интерфейс

##  Технологии

- **React** (хуки, состояние)
- **CSS Modules** (стилизация)
- **LocalStorage** (сохранение данных)
- **Vercel** (деплой)
